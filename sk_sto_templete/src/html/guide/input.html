component : input text

텍스트 입력창

텍스트상태

input 텍스트는 모두 사용 가능 하다

에러 메세지는 "helper-text"라는 클래스명으로 추가 하여 사용 한다.

에러 상황은 input에 "invalid" 클래스를 삽입 시 "helper-text"의 data-error이 화면에 표시가 된다.

달력은 bootstrap-datepicker3를 사용한다.

파일 첨부 시 서버에서 성공 메세지가 올 경우 "check"에 "active" 클래스를 삽입하면 인터렉션이 이루어진다.

textarea에서는 초기값은 작은 높이에서 점점 늘어나도록 구성을 한다
